## About This Course

This repository contains all the code examples, exercises, and projects from my JavaScript learning journey. Each topic is organized into its own section with clear examples and hands-on practice.

## Prerequisites

- Basic understanding of HTML & CSS
- A code editor (VS Code recommended)
- A modern web browser (Chrome, Firefox, Edge)
- Curiosity and a willingness to learn!

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR_USERNAME/javascript-course.git
   ```
2. **Navigate to the project folder**
   ```bash
   cd javascript-course
   ```
3. **Open in VS Code**
   ```bash
   code .
   ```
4. **Start learning!** Open any lesson folder and follow along.

## Course Outline

### Part 1: Fundamentals

| #  | Topic                          | Description                                      |
|----|--------------------------------|--------------------------------------------------|
| 01 | Introduction to JavaScript     | What is JS, how it works, and where to write it  |
| 02 | Variables & Data Types         | `var`, `let`, `const`, strings, numbers, booleans |
| 03 | Operators                      | Arithmetic, comparison, logical, and assignment   |
| 04 | Control Flow                   | `if/else`, `switch`, ternary operator             |
| 05 | Loops                          | `for`, `while`, `do...while`, `for...of`          |
| 06 | Functions                      | Declarations, expressions, arrow functions        |
| 07 | Arrays                         | Creating, accessing, and manipulating arrays      |
| 08 | Objects                        | Object literals, properties, methods              |
| 09 | Strings                        | String methods and template literals              |

### Part 2: Intermediate

| #  | Topic                          | Description                                      |
|----|--------------------------------|--------------------------------------------------|
| 10 | DOM Manipulation               | Selecting, modifying, and creating elements       |
| 11 | Events                         | Event listeners, event object, delegation         |
| 12 | Higher-Order Functions         | `map`, `filter`, `reduce`, `forEach`              |
| 13 | Destructuring & Spread         | Modern syntax for arrays and objects              |
| 14 | Error Handling                 | `try/catch`, throwing errors                      |
| 15 | Local Storage                  | Storing and retrieving data in the browser        |

### Part 3: Advanced

| #  | Topic                          | Description                                      |
|----|--------------------------------|--------------------------------------------------|
| 16 | Asynchronous JavaScript        | Callbacks, Promises, `async/await`                |
| 17 | Fetch API                      | Working with APIs and HTTP requests               |
| 18 | ES6+ Features                  | Modules, classes, symbols, iterators              |
| 19 | OOP in JavaScript              | Prototypes, classes, inheritance                  |
| 20 | Closures & Scope               | Lexical scope, closures, hoisting                 |

### Projects

| #  | Project                        | Concepts Covered                                 |
|----|--------------------------------|--------------------------------------------------|
| P1 | Calculator                     | DOM, Events, Functions                            |
| P2 | To-Do List                     | CRUD, Local Storage, DOM                          |
| P3 | Weather App                    | Fetch API, Async/Await, DOM                       |
| P4 | Quiz Game                      | Arrays, Objects, Events, Timers                   |
